🛡️ NerdForge AI

## Autonomous AI Security Operations Center

NerdForge AI is an AI-powered cybersecurity platform that combines attack simulation, SOC monitoring, incident response, and threat hunting into one intelligent ecosystem.

## Tech Stack

- **Frontend:** React + TypeScript + Vite + Tailwind CSS
- **Backend:** FastAPI + Python + SQLAlchemy
- **AI:** Google Gemini AI + LangChain + LangGraph
- **Database:** SQLite (development) / PostgreSQL (production)

## Setup Instructions

### Backend
\`\`\`bash
cd backend
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
uvicorn app.main:app --reload
\`\`\`

### Frontend
\`\`\`bash
cd frontend
npm install
npm run dev
\`\`\`
